# Ama3 Bot

A Discord bot written in Go, integrating OpenAI via the official openai-go client.

## Overview
Ama3-bot enables AI-driven interaction inside Discord using OpenAI.

Built in Go with openai-go, it focuses on simplicity, type safety, and efficient request handling.

## Features
- AI-powered responses via OpenAI
- Discord command handling
- Typed OpenAI client integration (openai-go)
- Lightweight and fast execution
- Extendable structure for additional features

## Tech Stack
- Go
- discordgo (Discord API)
- openai-go (official OpenAI client)
